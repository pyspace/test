Time Series object

Represents a finite length time series consisting (potentially) of several channels. Objects of this type are called “Windows” in other contexts.

The time series object is a 2d array of channels times time series amplitudes with some additional properties. The additional properties are:

  • channel names

  • sampling frequency

  • start time

  • end time

  • marker name (the name of the marker used to create this object,

    dictionary of included marker names and time stamps)

  • name & tag (text format of object meta info)

Channels can also be pseudo channels after spatial filtering.
